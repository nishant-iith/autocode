API Overview

The AutoCode API provides a complete backend solution for building online code editors and development environments. It supports project management, real-time file operations, template-based project creation, and seamless import functionality from various sources.

Key Endpoints

GET /api/health
Check API server health status
GET /api/projects/list
Retrieve all projects in workspace
POST /api/projects/create
Create new Node.js project
POST /api/projects/import-github
Import project from GitHub repository
GET /api/files/tree/{workspaceId}
Get complete file tree structure
GET /api/templates/list
List available project templates
